============================================================
  ANIMEWONDER — COMPLETE SETUP GUIDE
============================================================

STEP 1 — Install Python packages
---------------------------------
    pip install -r requirements.txt

  Also install ffmpeg (needed for video export):
    Download from: https://ffmpeg.org/download.html
    Extract and add the bin/ folder to your Windows PATH.
    (Or just pip install imageio-ffmpeg — it auto-downloads ffmpeg)


STEP 2 — Set environment variables
------------------------------------
Easiest: put them in a .env file next to app.py (already loaded automatically).
Or set them in your shell:

    set ANTHROPIC_API_KEY=your_anthropic_api_key
    set ADMIN_USER=admin
    set ADMIN_PASS=a_long_random_string     (REQUIRED — app refuses to start with
                                              defaults like "changeme", "admin",
                                              "password", "123456", or empty)
    set SECRET_KEY=any_random_long_string   (optional — if unset, the app auto-
                                              generates one and persists it to
                                              .flask_secret next to app.py so
                                              sessions survive restarts)

  For PayPal payments (to charge users):
    set PAYPAL_CLIENT_ID=your_paypal_client_id
    set PAYPAL_SECRET=your_paypal_secret
    set PAYPAL_MODE=sandbox              (DEFAULT — safe for testing)
                                          (Set to 'live' ONLY when going to
                                           production. Live mode requires
                                           CLIENT_ID and SECRET to be set or
                                           the app refuses to start.)
    set PAYPAL_PLAN_HUNTER=P-...         (your $9.99/mo PayPal plan ID)
    set PAYPAL_PLAN_MONARCH=P-...        (your $49.99/mo PayPal plan ID)


STEP 3 — Run
--------------
    python app.py

  Open:  http://localhost:5000

============================================================
  VOICE OVER
============================================================

  Voiceovers use the browser's built-in Speech Synthesis API.
  - Click the 🔇 VOICE button to enable
  - Each character gets a different voice automatically
  - Emotion affects speed and pitch (angry = faster, sad = slower)
  - Scene narration is read first, then dialogue plays in sequence
  - Works on Chrome, Edge, Safari, Firefox

  In exported VIDEO:
  - gTTS (Google Text-to-Speech) narrates each scene
  - Narrator reads: scene title, action description, then all dialogue
  - Audio is baked into the MP4

============================================================
  PAYPAL SETUP (to charge users)
============================================================

1. Create PayPal Business account at paypal.com/business
2. Go to: Developer Dashboard → My Apps → Create App
   Copy Client ID → set as PAYPAL_CLIENT_ID
   Copy Secret    → set as PAYPAL_SECRET
3. Go to: Subscriptions → Plans → Create Plan
   - Plan 1: $9.99/month → copy Plan ID → PAYPAL_PLAN_HUNTER
   - Plan 2: $49.99/month → copy Plan ID → PAYPAL_PLAN_MONARCH
4. Go to: Developer → Webhooks → Add Webhook
   URL: https://yourdomain.com/paypal-webhook
   Events: BILLING.SUBSCRIPTION.ACTIVATED
           BILLING.SUBSCRIPTION.CANCELLED
           BILLING.SUBSCRIPTION.SUSPENDED
5. Restart the app with the new env vars

  Users pay with their PayPal account (or card via PayPal).
  Payment verified server-side before tier upgrade.
  Cancellations automatically downgrade the user to Free.

  TEST FIRST: Use PAYPAL_MODE=sandbox with sandbox credentials.
